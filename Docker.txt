Че делать с докером:

Как обновить в репозитории:
1. Запушить последнюю версию ConnectorCenter в докер. 
    Для этого удалить контейнер из репозитория (или разобраться как обновлять поверх). Создать новый с именем "connectorcenter".
    Запушить из VS проект через "Опубликовать" (проверить что нет изменений на гите перед этим).

Как запустить из репозитория:
1. Запустить оркестратор. 
    Запускаем терминал от админа
    Берем план оркестра из VS (в проекте docker-compose файл docker-compose.yml)
    Сохраняем его в отдельную папку как файл с таким же именем .yml (или просто запоминаем где он локально находится)
    В терминале пишем 'cd <папка где лежит файл>'
    Далее пишем 'docker-compose up'

    Если пишет что не может найти докерфайл:
        'docker run avenusda/connectorcenter'
        Ждем пока запустится. Выключаем через интерфейс докера.
        И уже потом 'docker-compose up'
2. По пути localhost:8000 проверяем что все запустилось.
